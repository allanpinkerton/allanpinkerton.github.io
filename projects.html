Android

Helicon

Helicon is a to-do list like time management app on Android. The main feature that will be added to the app is the ability to "predict" how long certain tasks will take based on previous tasks. For example, if you took 3 hours to complete Math249 Assignment 1 and 2 hours to complete Math249 Assignment 2, then the program would guess that Math249 Assignment 3 would take a similar amount of time (2-3 hours). Due to the sheer size and amount of work required, the project has been put on hold either until I could find someone else to cooperate with or until I have more spare time.

If you are interested, the source code is available on Github, with more details on which features still need to be implemented.
